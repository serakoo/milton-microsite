To build on that

dispatch builds on Bootstrap. You can build on dispatch.

dispatch builds on Bootstrap, so a lot of CSS and general usage is approachable to anyone, and especially to those that are already familiar with Bootstrap framework. You are welcome to edit and improve dispatch, but remember to share back the evolutions on our IDEO Github / dispatch repository.



You can extend the features and components of dispatch using Bootstrap elements:

  1. For more info on CSS http://getbootstrap.com/css/
  2. For more info on Icons and Components http://getbootstrap.com/components/
  3. For more info on Javascript and Bootstrap http://getbootstrap.com/javascript/.
