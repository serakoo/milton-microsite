Setup & Usage

Instructions on how to get started and edit content

Here is all you need to know on how to use dispatch and customize it.

Folders and files

  • /pages: where all the content pages are stored.
  • /css: where all the stylesheets can be found
  • /css/dispatch.css: the main stylesheet you might want to start from to customize dispatch.
  • /img: images folder, where the "My Client + IDEO" logo is also stored.

Advanced: Folders and files

  • index.php: main container / page structure.
  • /includes/modal.html: is the template for modals
  • /includes/functions.php: where all the PHP scripts are stored
  • /js: where all the Javascripts are stored
  • /js/dispatch.js: the main Javascript you might want to start from to extend behaviors

To edit the content of the menu (and the site content)

  1. Open menu.json in the /js folder
  2. Edit structure and content of you menu / microsite from here
  3. All section names must correspond to a .html file saved in the /pages folder

Syntax:
{"Name of the section": "file-name-of-the-section-without-the-extension"}

Example:
{"Setup & Usage": "how-to"}
which corresponds to a file named how-to.html stored into the /pages folder

To run the microsite on your local computer

  1. Open the Terminal
  2. cd to the directory of the microsite: cd /Documents/Users/Myself/YOUR_FOLDER/
  3. Paste this command and press enter: php -S localhost:7777
  4. Open the browser with this URL: http://localhost:7777

To publish your microsite, you'd need

  1. A web server configured with PHP (all IDEO servers are ready to run PHP)
  2. That's it
